# ADRES Frontend

Este proyecto fue generado con [Angular CLI](https://github.com/angular/angular-cli).

## Descripción

Esta es la aplicación frontend para el sistema de adquisiciones de ADRES.

## Requisitos Previos

Asegúrate de tener instalado Node.js y Angular CLI.

-   [Node.js](https://nodejs.org/) (Se recomienda la versión LTS)
-   [Angular CLI](https://cli.angular.io/): `npm install -g @angular/cli`

## Instalación

1.  Clona el repositorio.
2.  Navega a la carpeta `frontend`.
3.  Instala las dependencias del proyecto:

    ```bash
    npm install
    ```

## Servidor de Desarrollo

Ejecuta `ng serve` para un servidor de desarrollo. Navega a `http://localhost:4200/`. La aplicación se recargará automáticamente si cambias alguno de los archivos de origen.

```bash
ng serve
```

## Compilación (Build)

Ejecuta `ng build` para compilar el proyecto. Los artefactos de compilación se almacenarán en el directorio `dist/`.

```bash
ng build
```

## Más Ayuda

Para obtener más ayuda sobre Angular CLI, usa `ng help` o visita la documentación de Angular CLI.
